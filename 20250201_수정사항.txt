*차트
1. 차트에서 y축은 달러기준
2. 라이너 차트
3. 가격 실시간
4. 까딱까딱
5. 1D 1W 1M 3M ALL
6. 거래량 표시(달러 기준)
7. NFT 그림 표시해주기
8. settle 카운트다운 넣기

*오더북
1. 오더북 수량에서 소수점 빼기
2. 블록배경 넣기
3. 가격 / 수량 제목 넣기
4. 현재 구매자 / 판매자 비율 => 게이지 안에 % (펀딩피 같은 UI)

*주문창
1. 바이/셀 버튼
	- 돈이 충분하다면 => BUY / SELL
	- 돈이 없다면 => Insufficent balance
2. BUY/ SELL 누르면 모달창 없음, 바로 메타마스크 열림
	- 이 때 total만큼 디파짓 함

*바텀탭
1. 포지션 목록 / 오더 목록 / 히스토리
2. 포지션 목록 = 체결 된 주문
3. 오더 목록 = 미체결 된 주문
4. 히스토리 = 미체결 주문, 체결 주문, 취소 주문
